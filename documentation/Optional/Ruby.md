# Ruby

## Multiple Ruby versions
Install rbenv to support multiple Ruby versions:

```bash
brew install rbenv
```

Instal the default 2.x version:

```bash
$ rbenv install -f 2.3.0
```

Set the newly installed version as the default:

```bash
$ rbenv global 2.3.0
```


## Bundler
Install bundler:

```bash
$ gem install bundler
```
